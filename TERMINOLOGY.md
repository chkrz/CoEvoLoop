# CoEvoLoop 项目术语表

## 核心概念术语

### 数据集相关
| 中文 | 英文 | 说明 |
|---|---|---|
| 数据集 | Dataset | 用于训练和评测的对话数据集 |
| 标注 | Annotation | 对对话数据进行人工标记和分类 |
| 合成 | Synthesis | 通过AI生成新的对话数据 |
| 评测 | Evaluation | 对模型性能进行评估和打分 |
| 数据清洗 | Data Cleaning | 清理和标准化原始数据 |
| 数据验证 | Data Validation | 检查数据质量和格式正确性 |

### 强化学习相关
| 中文 | 英文 | 说明 |
|---|---|---|
| 强化学习 | Reinforcement Learning | 通过奖励机制训练AI模型的方法 |
| 奖励 | Reward | 对模型行为的正向或负向反馈 |
| 奖励模型 | Reward Model | 用于计算奖励的神经网络模型 |
| 推演 | Rollout | 模拟对话过程以收集训练数据 |
| 策略 | Policy | 模型决定如何响应的策略网络 |
| 价值函数 | Value Function | 评估状态价值的函数 |
| PPO | Proximal Policy Optimization | 近端策略优化算法 |

| 提示词 | Prompt | 指导AI模型生成响应的文本 |
| 系统消息 | System Message | 定义AI助手行为的指令 |

## 使用说明

1. **严格遵循术语表**：所有翻译必须使用本术语表中定义的标准术语
2. **保持一致性**：同一术语在整个项目中必须保持相同的翻译
3. **上下文适配**：根据具体语境选择最合适的术语
4. **定期更新**：随着项目发展，及时更新和扩展术语表
5. **代码审查**：代码审查时必须检查术语使用的一致性
